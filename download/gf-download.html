Requirements

Linux, Windows, Mac OS X, or Solaris

GHC (Glasgow Haskell Compiler)

For the GUI: Java 1.4

Download

GF-2.0.tgz, source and grammar package

Compile and install

Uncompress: 
  tar xvfz GF-2.0.tgz
Follow instructions in GF-2.0/INSTALL

Run

Without the GUI: 
  gf
With the GUI: 
  jgf
